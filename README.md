# 🇦🇺 Australian Job Market Analytics Dashboard

An interactive dashboard analysing 5,000+ Australian job postings to identify in-demand roles, salary benchmarks, hiring trends, and top companies across Australian cities.

## Built With
- Python
- Streamlit
- Pandas
- Plotly

## Features
- Top in-demand job titles across Australia
- Job postings by city (Melbourne, Sydney, Brisbane, Perth, Adelaide, Canberra)
- Salary benchmarks by role
- Work type breakdown
- Top hiring companies
- Industry distribution
